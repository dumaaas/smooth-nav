Logos

Column Five’s logo was created using an inverted triangle as the template. The inverted triangle represents a “V” for the roman numeral “5” and the word “visual.”

Color

Column Five’s color primary color palette consists of black and white, with red accents as needed. Consistent color use strengthens our brand identitiy.

Type

Column Five’s brand typeface is Hurme Geometric Sans. To add flexibility, the full Hurme family can be used in any materials.
